# Лабораторная работа №1: Перемножение матриц

## Описание работы

Реализация алгоритма перемножения матриц с использованием:
- **CPU**: алгоритм с перестановкой циклов (i-k-j)
- **GPU**: параллельная реализация на CUDA

## Особенности реализации

На CPU использовался классический алгоритм с троиным вложенным циклом, который работает за O(N³)

На GPU каждый поток вычисляет один элемент результирующей матрицы. При этом тысячи потоков работают одновременно

## Анализ результатов

1. **Для матриц (≤250×250)**: ускорение незначительное из-за накладных расходов, поэтому применять распараллеливание нецелесообразно
2. **Для матриц (500×500)**: ускорение достигает 27x
3. **Для матриц (≥1000×1000)**: ускорение варьируется на уровне ~30-32x
