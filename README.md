# Билатеральный фильтр: CPU vs GPU (CUDA)

## Описание работы

Реализация билатерального фильтра для обработки изображений с использованием:
- **CPU**: последовательная реализация на C++
- **GPU**: параллельная реализация на CUDA

## Задача

Билатеральный фильтр — это нелинейный фильтр, который сглаживает изображение, сохраняя при этом резкие границы (края). Он использует комбинацию:
- **Пространственного веса** (Gaussian) — учитывает расстояние до центрального пикселя
- **Диапазонного веса** — учитывает разницу в интенсивности


Для тестирования использовалось синтетическое изображение размером 256×256 пикселей, содержащее:
- Градиентный фон
- Белый и черный квадраты
- Круг
- Линии (крест)
- Искусственный шум
